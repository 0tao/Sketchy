Pathing Processor

Pathing Processor Options can be selected to tell the processor in which order to search for adjacent pixels and other pathing behaviors.

Pattern

Follow Left
Checks for pixels 90 degrees left of the current direction of travel, continuing clockwise until a pixel is found to draw.
Follow Right
Checks for pixels 90 degrees right of the current direction of travel, continuing counter-clockwise until a pixel is found to draw.

End of Line Behavior

Backtrack (** Recommended **)
As pixels are drawn, they are stored in memory and when no adjacent pixels are found, retrace the line backwards checking for any other adjacent pixels.
Lift Pen
Instead of backtracking, just lift the pen to the next pixel. May be beneficial with some image types, but typically lifts the pen excessively.

WARNING: Rapid, repetitive pen lifts can burn out a servo!!

Vertical Striping Length

Used to draw the image in striped sections from the top to the bottom
Set this value to the length in millimeters for the length of each Stripe, or 0 for no striping.
Typically used for Dry Erase drawings so the Pen Holder bolt heads don't erase the picture while drawing.
Note: Minor spaces/lines at stripe borders can occur depending on QuickDraw alignment or "play" in the pulleys and/or strings.
