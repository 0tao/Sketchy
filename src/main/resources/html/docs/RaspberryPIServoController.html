Raspberry PI Servo Controller

Left/Right Motor Properties

The default values should be correct if you are using a QuickDraw controller board and connected it according to the instructions.

Use the "Motor Forward" and "Motor Backward" buttons to test the Left and Right Motors.

If the motors are swapped (left/right), then you can either: shutdown and power off the QuickDraw and swap the connectors; or you can swap the pin numbers on this screen, but make sure to swap all pins correctly.

"Motor Forward" should extend the string, and "Motor Backward" should retract it.

If "Motor Forward" retracts instead of extends, you can either: wind the pulleys in the opposite direction; or shutdown and power off the QuickDraw and reverse the connector(s); or you can select/deselect the "Invert Direction" check box for the motor(s).

Pen Properties

The default Servo Device Pipe and Pen Pin Number should be correct if you are using a QuickDraw controller board and connected it according to the instructions.

When calibrating the servo, the goal is to use the smallest amount of travel as needed for proper operation. The servo should not meet any resistance other than what it takes to move the Pen. Too much resistance can overheat or strip the gears on the servo!

Use the "Pen Up", "Pen Down", and "Cycle 5 Times" to test the servo operation.

The next four properties are critical for proper Pen servo Operation

*** Quick and/or Repetitive movements of the servo can severely reduce it's lifespan! ***

Up Stable Period: The amount of time (in milliseconds) to allow the pen to lift from the drawing surface before moving.

Down Stable Period: The amount of time (in milliseconds) to allow the pen to return to the drawing surface before drawing.

Pen Up Position: The position of the servo to put the pen down.

Pen Down Position: The position of the servo to lift the pen up.
