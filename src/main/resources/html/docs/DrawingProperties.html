Drawing Properties

Default Pen Width

Select the Default Pen Width that shown on the Render page when first starting up the QuickDraw.

Drawing Sizes

Drawing Sizes can be defined and are shown in the drop down list on the Render page.

Each Drawing size is defined by a width x height (in mm),
For example: "200x250"

Text can also be added to the end of the Drawing size to help describe it,
For example. "200x250 (Letter-Portrait)"
